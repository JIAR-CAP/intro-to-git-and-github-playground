# intro-to-git-and-github-playground

## Requirements

Install [Ruff](https://docs.astral.sh/ruff/installation/) (linter and formatter):

```
conda install -c conda-forge ruff
```

## Run

```
$ python app.py 10 10
10 + 10 = 20
```

## Test

```
$ python app_test.py
.
----------------------------------------------------------------------
Ran 1 test in 0.000s

OK
```

## Lint & Format

```
ruff check --fix
ruff format
```
